
# Kleio notation for historical sources

`Kleio` is a notation for the transcription of historical sources developed by Manfred Thaller.

Its aim is to transcribe the relevant information from a Historical Source in a form close to the original in terms of the sequence and structure of the information. A Kleio transcription of a historical source is _readable_, can preserve original orthography if needed, include comments by the transcriber,
